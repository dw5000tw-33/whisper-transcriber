# Whisper語音轉文字

一個使用 OpenAI Whisper 的桌面語音轉文字工具。  
支援本機音檔與 YouTube 網址轉錄，並可輸出 `.txt` 文字檔。

## 功能

- 支援音檔轉文字
- 支援 YouTube 網址下載後轉文字
- 可選 Whisper 模型
- 可選語言或自動偵測
- 轉錄結果可另存文字檔
- 提供進度提示與 loading 狀態

## 安裝需求

建議使用：

- Python 3.10 或 3.11
- ffmpeg
- Node.js

## 安裝 Python 套件

```bash
pip install -r requirements.txt
